¿Qué es un hosting?

El hosting es el servicio dado por una empresa que se ocupa de almacenar toda la información que colocas en una página web. Cuando creas una página web, su diseño y sus elementos (como información, plantillas, imágenes, video...) deben ubicarse en algún lugar de Internet. El servicio de ubicación y almacenamiento es el hosting y hay empresas que se dedican exclusivamente a ofrecerlo. Este consiste en que te alquilan espacio en el disco duro de uno de sus servidores web (computadoras de gran capacidad especializadas para esta tarea) y accesos de alta velocidad para todos los usuarios de Internet que requieran observar la información de tu sitio. La lógica de funcionamiento consiste en que un usuario digita tu dirección web en un explorador, el servicio DNS envía al servidor o hosting y este muestra tu página web.

¿Qué es un dominio?

Un dominio o nombre de dominio es la dirección de tu sitio web. Cada dominio es único en Internet, por lo que no pueden existir dos sitios con el mismo dominio. Es semejante a www.compuhouseweb.net: esa dirección solo te puede llevar a una página, que es la de esa empresa, y a ninguna otra. El nombre de dominio está asociado con una dirección IP de Internet. Cada página tiene asignado un número específico. Una dirección IP está construida por una serie lógica de números, por ejemplo: 167.305.257.299. Podemos suponer que esta dirección nos lleva a www.compuhouseweb.net. Para la mayoría de los usuarios de Internet, es más fácil recordar el nombre del sitio que la dirección IP.

Diferencias entre hosting y dominio

DOMINIOHOSTING
Es el nombre de un sitio web, la dirección de Internet que deben introducir los usuarios en sus navegadores para acceder a este. Es el espacio en un determinado servidor donde se guardan todos los archivos que forman el sitio web.
Cuando adquieres un dominio web, estás comprando el derecho, en exclusiva a utilizar esa dirección. El servidor es administrado por una empresa de hosting o por un proveedor de creadores web.
El dominio suele estar formado por tres partes:
  • www (está presente siempre, o casi siempre, en todos los dominios)
  • Nombre de dominio ("miempresa", "useit", etc.)
  • Extensión de dominio (".com", ".es", ".cat", etc.)
 Podemos encontrar 3 tipos de hosting:
  • Hosting compartido: proporciona un servidor en el que se comparten recursos con más usuarios.
  • Hosting dedicado: se trata de una máquina física asignada en exclusiva a un único cliente.
  • Hosting dedicado administrado: el usuario no se tiene que preocupar por la gestión y administración del mismo, lo realiza el servicio de hosting.
Para elegir el nombre de dominio, es necesario considerar las palabras clave del sitio, la longitud del nombre y su capacidad de ser recordado por los usuarios. Se debe considerar que la información siempre esté disponible, no debería haber cortes o caídas en el sitio web.
Los beneficios de tener un dominio son:
  • Resalta la innovación y profesionalismo de la empresa.
  • Los clientes pueden recordar la empresa facilmente.
  • Los buscadores de Google lo incluyen en el ranking.
  • Es una forma de garantizar y proteger el nombre de una marcar.
  • Se habilita la posibilidad de tener un correo empresarial con el nombre del dominio.
 Aún cuando un hosting es de paga, y se obtiene uno gratuito, se puede beneficiar de la siguiente manera:
  • No se necesita pagarlo, así que se puede destinar el presupuesto para otras necesidades.
  • Existen lugares profesionales para tener una página gratis. Hay plataformas como WordPress o Blogger que permiten crear una página web o blog gratis.
  • Se puede alojar archivos HTML y CSS en el hosting gratis, y guardar imágenes, videos y archivos multimedia en otra ubicación para ocupar menos espacio.
